# CloudSim-Benchmark

基于CloudSim Plus的云调度算法性能评估框架

## 🚀 快速开始

### 运行批处理实验
```bash
./run batch
```

### 运行实时调度实验
```bash
./run realtime
```

### 构建项目
```bash
./run build
```

## 📖 详细文档

请查看 [`docs/README.md`](docs/README.md) 获取完整的使用说明和API文档。

## 📁 项目结构

- `configs/` - 配置文件目录
- `src/` - 源代码
- `scripts/` - 构建和运行脚本
- `docs/` - 详细文档
- `tools/` - 工具和可视化脚本
- `data/` - 数据文件目录

## 📊 实验结果

实验结果保存在 `runs/` 目录下，采用YOLO风格的目录结构。
